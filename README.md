# Codeforces Contest Analyzer

A full-stack web application for analyzing Codeforces contest performance, rating progression, and competitive programming insights through interactive visualizations.

## Features

- Search and analyze Codeforces users by handle
- Fetch real-time contest and profile data using the Codeforces API
- Visualize rating progression and contest performance trends
- Interactive dashboard with charts and analytics
- Backend REST APIs for contest data processing and analysis
- Persistent database storage for efficient data handling

## Tech Stack

**Frontend**
- React.js

**Backend**
- Python
